================================================================================
                         1. PROJECT OVERVIEW
================================================================================

PROJECT 159 is a comprehensive biblical cross-reference visualization tool that
maps the textual relationships between 159 books across all major biblical
traditions. The project expands beyond the traditional 66-book Protestant canon
to include:

  * 66 Protestant Canon Books (OT 39 + NT 27)
  * 14 Deuterocanonical Books (Catholic/Orthodox additions)
  * 21 Ethiopian/Pseudepigraphal Texts
  * 10 Dead Sea Scrolls
  * 22 Gnostic & Early Christian Writings
  * 26 Lost Books (Referenced in Scripture but lost to history)

The name "PROJECT 159" reflects the 159 books tracked across all biblical
traditions: 66 Protestant + 14 Deuterocanonical + 21 Ethiopian + 10 DSS + 22 Gnostic + 26 Lost.

================================================================================
                         2. MISSION & PURPOSE
================================================================================

PRIMARY GOALS:
--------------
1. TRUTH SEEKING: Provide an objective, data-driven view of biblical textual
   relationships without denominational bias

2. EDUCATION: Help users understand how different biblical traditions relate
   to one another through shared themes, quotations, and allusions

3. TRANSPARENCY: Show which books quote or reference which other books,
   including controversial non-canonical texts

4. HISTORICAL CONTEXT: Present the complex history of biblical canonization
   and why different traditions accept different books

5. ACCESSIBILITY: Make advanced biblical scholarship accessible to everyone
   through interactive visualizations

WHY THIS MATTERS:
-----------------
Most people only know the 66-book Protestant Bible. However:
- Catholics include 7 additional books (73 total)
- Orthodox churches include even more books
- Ethiopian Orthodox has the largest canon (81+ books)
- The New Testament quotes books not in any modern Bible
- Early Christians read many texts later rejected

Understanding these connections reveals the complex, human process of
deciding what counts as "scripture" - a process still debated today.

                         4. TECHNICAL ARCHITECTURE
================================================================================

TECHNOLOGY STACK:
-----------------
- HTML5/CSS3 for structure and styling
- JavaScript (ES6+) for interactivity
- D3.js v7 for data visualization
- Local JSON data files (no external API dependencies)
- GitHub Pages for hosting

KEY JAVASCRIPT MODULES:
-----------------------
1. data-loader.js
   - Loads and parses 88-book base data
   - Handles progress updates during loading
   - Provides filtering and statistics functions

2. data-augmenter-147.js
   - Augments base 88 books to full 147+ book dataset
   - Adds Dead Sea Scrolls, Gnostic, and Lost books
   - Creates chapter-level data structures

3. arc-diagram-tableau-style.js
   - Renders the arc diagram visualization
   - Implements rainbow color coding by distance
   - Draws non-canonical cross-reference arcs (red, dark red, gold)
   - Handles tooltips and interactivity

4. cross-ref-loader.js
   - Loads cross-reference data from text files
   - Categorizes references by type (Canonical, Ethiopian, etc.)
   - Generates descriptions for connections

5. theographic-loader.js
   - Loads supplementary data about biblical entities
   - Supports Bible text lookups for verse content

================================================================================
                         5. DATA SOURCES & METHODOLOGY
================================================================================

PRIMARY DATA SOURCES:
---------------------
1. Cross-References Database
   - Over 344,000 canonical cross-references
   - Derived from Treasury of Scripture Knowledge
   - Enhanced with community-voted connections

2. Non-Canonical References
   - Manually curated from scholarly sources
   - Includes academic consensus on quotations/allusions
   - Weighted by confidence level (100=high, 50=medium, 25=lower)

3. Book Information
   - Ethiopian Orthodox Tewahedo Bible canon
   - Catholic Deuterocanon
   - Dead Sea Scrolls scholarly editions
   - Nag Hammadi Library texts

VOTE/WEIGHT SYSTEM:
-------------------
- 100 votes = Direct quotations or explicit citations
  Example: Jude 1:14-15 quoting 1 Enoch 1:9

- 50 votes = Strong thematic or conceptual parallels
  Example: Matthew 25:31 and 1 Enoch 62 (Son of Man imagery)

- 25 votes = Possible allusions or shared themes
  Example: John 5:22 and 1 Enoch 69:27 (judgment given to Son)

================================================================================
                         6. VISUALIZATION FEATURES
================================================================================

ARC DIAGRAM:
------------
- Chapters arranged horizontally across the full biblical sequence
- Arcs connect chapters with cross-references
- Rainbow gradient shows distance (red=close, violet=far)
- Non-canonical arcs highlighted with special colors:
  * RED: Deuterocanonical & Ethiopian references
  * DARK RED: Gnostic/Early Christian references
  * GOLD: Lost/Referenced book citations

INTERACTIVE FEATURES:
---------------------
- Hover over arcs to see connection details
- Hover over book labels for highlighting
- Filter by testament (OT only, NT only, cross-testament)
- Adjust minimum connection threshold
- Multiple color themes available

SORTABLE TABLES:
----------------
- Click column headers to sort
- Filter by reference type (tabs)
- Search by book name or reference
- Expandable verse text when available

                         11. GNOSTIC TEXTS ANALYSIS
================================================================================

GOSPEL OF JUDAS - A CASE STUDY IN SCHOLARLY CAUTION:
----------------------------------------------------
The Gospel of Judas (published 2006) provides an instructive example:

1. MEDIA HYPE vs REALITY
   - National Geographic marketed it as "rewriting Christian history"
   - Actual content is late 2nd century Gnostic fiction
   - No historical information about the real Judas

2. TRANSLATION ERRORS (Later Corrected)
   - Initial: "You will exceed all of them. For you will sacrifice the
     man that clothes me."
   - Corrected: "You will do worse than all of them. For the man that
     clothes me, you will sacrifice him."
   - The "hero Judas" narrative was based on mistranslation

3. GNOSTIC CONTEXT
   - Written by Sethian Gnostics (hostile to orthodox Christianity)
   - Demiurge theology: creator god is evil
   - Jesus' body = prison requiring destruction
   - Not a "lost gospel" but a sectarian polemic

4. LESSONS FOR BIBLICAL RESEARCH
   - Sensational claims require critical examination
   - Original language expertise matters
   - Historical context determines interpretation
   - "Ancient" doesn't mean "authentic" or "suppressed truth"
